rayslash v0.2.0 refines the launcher and makes the complete module-based build available across the major portable Linux formats.

Highlights:

- Redesigned module management, search-engine, alias, and diagnostics settings surfaces.
- Persistent in-window editors for aliases and search engines, with safer explicit deletion.
- High-quality cached favicons in settings and search results.
- Faster resident shortcut toggling and no optional module work for empty searches.
- RPM, DEB, AppImage, and Flatpak downloads for x86_64 and ARM64.

The RPM download uses the matching `rayslash-module-host` RPM from this release. DEB, AppImage, and Flatpak downloads already include the module host. Optional modules remain uninstalled until selected in Settings.

Use `SHA256SUMS` to verify downloads. See the [installation guide](https://github.com/rslauncher/rayslash/blob/v0.2.0/docs/INSTALL.md) for format-specific installation and shortcut instructions.
